== Description ==
**Woocommerce Parsian Bank Gateway**

 با این افزونه می توانید ووکامرس خود را به درگاه بانک پارسیان متصل نمایید

این افزونه با آخرین تغییرات درگاه پارسیان سازگار می باشد.

درگاه پارسیان طی این سالها تغییرات زیادی در نحوه ی ارتباط  داشته و همین امر باعث شده بسیاری از پلاگین های پرداخت پارسیان که پشتیبانی نمیشدند از کار بیفتند.
که این نبود پلاگین رایگان پارسیان باعث فروش پلاگین این درگاه شده و در زمان نگارش این مطلب پلاگین این درگاه تا قیمت های ۵۰ هزارتومان هم به فروش میرسه.

ما در <a href="https://dpsoft.ir/%D9%88%D8%A8%D9%84%D8%A7%DA%AF/26/%D8%A7%D8%B3%D8%AA%D9%81%D8%A7%D8%AF%D9%87-%D8%A7%D8%B2-%D8%AF%D8%B1%DA%AF%D8%A7%D9%87-%D9%BE%D8%B1%D8%AF%D8%A7%D8%AE%D8%AA-%D8%A8%D8%A7%D9%86%DA%A9-%D9%BE%D8%A7%D8%B1%D8%B3%DB%8C%D8%A7%D9%86-%D8%AF%D8%B1-%D8%B2%D8%A8%D8%A7%D9%86-PHP" target="_blank">دی پی سافت</a> سال گذشته پکیج ارتباط این درگاه رو در مخزن <a href="https://packagist.org/packages/dpsoft/parsian-payment">کامپوزر</a> قرار دادیم و امسال هم با استفاده از این پکیج ، پلاگین ووکامرس رو تهیه کردیم و امروز به صورت رایگان در اختیار دوستان قرار میدیم.

== Installation ==
1. از نصب بودن افزونه‌ی soap مرتبط با php اطمینان داشته باشید.
2. پوشه `parsian-woocommerce` را در مسیر `/wp-content/plugins/` آپلود کنید
3. افزونه را از طریق منوی 'افزونه ها' در وردپرس فعال کنید
4. تنظیمات افزونه را می توانید از طریق قسمت تنظیمات ووکامرس / پرداخت انجام دهید
